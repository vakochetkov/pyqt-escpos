# pyqt-escpos
Tiny GUI for python-escpos tested with QR701 receipt printer

### How to run app

`python app.py`

### Dependencies

GUI:

`PyQt5>=5.15.0`

python-escpos:

`Pillow == 8.2.0`
`PyYAML == 5.4.1`
`appdirs == 1.4.4`
`argcomplete == 1.12.3`
`pyserial == 3.5`
`pyusb == 1.2.1`
`qrcode == 7.3.1`
`six == 1.15.0`
